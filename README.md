## ▶️ Start the project
- ### **Requirements**
  - **Node.js**;
  - **Git**
  - Package manager **npm** or **yarn**
​

Then clone the repository
​
```sh
  $ git clone https://github.com/DarioCostaAlves/TenziesGame.git
```
And then:
```sh
  # Open project folder
  $ cd tenzies
  # Install dependencies
  $ npm or yarn install
  # Start the application
  $ npm start
```
